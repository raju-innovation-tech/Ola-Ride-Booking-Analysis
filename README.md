# Ola Ride Booking Analysis

## Project Overview
This project analyzes Ola ride booking data using SQL and Power BI to uncover booking trends, customer behavior, cancellation patterns, revenue insights, and vehicle performance. The objective is to transform raw ride data into meaningful business insights through data analysis and interactive visualizations.

## Tools & Technologies
* SQL
* Power BI
* Microsoft Excel

## Key Analysis Performed
* Total Bookings Analysis
* Successful vs Cancelled Rides
* Booking Value Analysis
* Vehicle Type Performance
* Customer Booking Trends
* Driver Cancellation Analysis
* Revenue Insights
* Customer and Driver Ratings Analysis

## Dashboard Features
* Interactive Filters and Slicers
* KPI Cards
* Booking Status Visualization
* Vehicle-wise Performance Analysis
* Revenue and Cancellation Insights
* Customer Behavior Analysis

## Project Outcome
The dashboard provides actionable insights into ride booking operations, helping stakeholders understand customer preferences, ride performance, cancellation reasons, and overall business trends for better decision-making.
